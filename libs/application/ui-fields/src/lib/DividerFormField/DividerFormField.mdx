# DividerFormField

### Usage in a template

You can create a DividerFormField using the following function `buildDividerField`.

<Source language="typescript" code={`buildDividerField({})`} />

The previous configuration object will result in the following component:

<Canvas>
  <Story of={DividerFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<DividerFormField field={...} />` with the configuration object defined above.
