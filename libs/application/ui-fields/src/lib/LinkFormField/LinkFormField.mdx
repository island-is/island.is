# LinkFormField

### Usage in a template

You can create a LinkFormField using the following function `LinkFormField`.

<Source
  language="typescript"
  code={dedent(`
    LinkFormField({
      id: 'field.id',
      title: 'Download PDF',
      s3Key: 'lorem_ipsum_pdf_key',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={LinkFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<LinkFormField field={...} />` with the configuration object defined above.
