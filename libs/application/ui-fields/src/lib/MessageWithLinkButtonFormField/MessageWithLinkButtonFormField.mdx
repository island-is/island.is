# MessageWithLinkButtonFormField

### Usage in a template

You can create a MessageWithLinkButtonFormField using the following function `MessageWithLinkButtonFormField`.

<Source
  language="typescript"
  code={dedent(`
    MessageWithLinkButtonFormField({
      id: 'field.id',
      title: '',
      url: '/minarsidur',
      message: 'Go to my pages',
      buttonTitle: 'Áfram'
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={MessageWithLinkButtonFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<MessageWithLinkButtonFormField field={...} />` with the configuration object defined above.
