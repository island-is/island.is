# TitleFormField

### Usage in a template

You can create a TitleFormField using the following function `buildTitleField`.

<Source language="typescript" code={`buildTitleField({})`} />

The previous configuration object will result in the following component:

<Canvas>
  <Story of={TitleFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<TitleFormField field={...} />` with the configuration object defined above.
