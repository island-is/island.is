# SubmitFormField

### Usage in a template

You can create a SubmitFormField using the following function `buildSubmitField`.

<Source
  language="typescript"
  code={dedent(`
    buildSubmitField({
      id: 'field.id',
      title: 'Field title',
      actions: [
        {
          event: 'SUBMIT',
          name: 'Submit',
          type: 'primary',
        },
      ],
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={SubmitFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<SubmitFormField field={...} />` with the configuration object defined above.
