# FileUploadFormField

### Usage in a template

You can create a FileUploadFormField using the following function `buildFileUploadField`.

<Source
  language="typescript"
  code={dedent(`
    buildFileUploadField({
      id: 'field.id',
      introduction: 'Field introduction',
      maxSize: 10000000, // 10 MB
      uploadAccept: '.pdf',
      uploadHeader: 'Field upload header',
      uploadDescription: 'Field upload description',
      uploadButtonLabel: 'Field upload button label',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={FileUploadFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<FileUploadFormField field={...} />` with the configuration object defined above.
