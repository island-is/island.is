# ImageFormField

### Usage in a template

You can create a ImageFormField using the following function `buildImageField`.

<Source
  language="typescript"
  code={dedent(`
    buildImageField({
      id: 'field.id',
      title: 'Field title',
      image: 'https://images.ctfassets.net/8k0h54kbe6bj/019iw5SSHsBi5vnjxLYupo/b869ce9a3fcbd5060084b2a3a967005b/LE_-_Retirement_-_S1.svg',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={ImageFormFieldStories.Default} />
</Canvas>

By default the image takes up all of it's container's width, alternatively you can set the `imageWidth` to 'auto'.

<Source
  language="typescript"
  code={dedent(`
    buildImageField({
      id: 'field.id',
      title: 'Field title',
      imageWidth: 'auto',
      image: 'https://images.ctfassets.net/8k0h54kbe6bj/019iw5SSHsBi5vnjxLYupo/b869ce9a3fcbd5060084b2a3a967005b/LE_-_Retirement_-_S1.svg',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={ImageFormFieldStories.AutoWidth} />
</Canvas>

The field also supports using a static SVG import:

<Source
  language="typescript"
  code={dedent(`
    import { ReactComponent as Logo } from './logo.svg'

    buildImageField({
      id: 'field.id',
      title: 'Field title',
      image: Logo,
    })`)}
/>

You can also use this field into a custom component by using `<ImageFormField field={...} />` with the configuration object defined above.
