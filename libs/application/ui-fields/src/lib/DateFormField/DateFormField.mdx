# DateFormField

### Usage in a template

You can create a DateFormField using the following function `buildDateField`.

<Source
  language="typescript"
  code={dedent(`
    buildRadioField({
      id: 'field.id',
      title: 'Field title',
      placeholder: 'Select a date',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={DateFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<DateFormField field={...} />` with the configuration object defined above.
