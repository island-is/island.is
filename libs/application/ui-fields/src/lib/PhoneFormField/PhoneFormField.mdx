# PhoneFormField

### Usage in a template

You can create a PhoneFormField using the following function `buildPhoneField`.

<Source
  language="typescript"
  code={dedent(`
    buildPhoneField({
      id: 'field.id',
      title: 'Field title',
      placeholder: 'Enter your email',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={PhoneFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<PhoneFormField field={...} />` with the configuration object defined above.
