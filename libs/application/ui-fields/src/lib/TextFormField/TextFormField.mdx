# TextFormField

### Usage in a template

You can create a TextFormField using the following function `buildTextField`.

<Source
  language="typescript"
  code={dedent(`
    buildTextField({
      id: 'field.id',
      title: 'Field title',
      placeholder: 'Enter your email',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={TextFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<TextFormField field={...} />` with the configuration object defined above.
