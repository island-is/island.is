# SelectFormField

### Usage in a template

You can create a SelectFormField using the following function `buildSelectField`.

<Source
  language="typescript"
  code={dedent(`
    buildSelectField({
      id: 'field.id',
      title: 'Field title',
      options: [
        {
          value: '20',
          label: '20%',
        },
        {
          value: '40',
          label: '40%',
        },
        {
          value: '60',
          label: '60%',
        },
        {
          value: '80',
          label: '80%',
        },
      ],
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>

      }}

  </Story>
</Canvas>

You can also use this field into a custom component by using `<SelectFormField field={...} />` with the configuration object defined above.
