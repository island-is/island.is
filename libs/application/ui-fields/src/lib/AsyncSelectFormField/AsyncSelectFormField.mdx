# AsyncSelectFormField

### Usage in a template

You can create a AsyncSelectFormField using the following function `buildAsyncSelectField`.

<Source
  language="typescript"
  code={dedent(`
    buildAsyncSelectField({
      id: 'field.id',
      title: 'Field title',
      isSearchable: true,
      loadOptions: async ({ apolloClient }) => {
        // Fetch something with apolloClient
        return await Promise.resolve([
          { label: 'Apple', value: 'apple' },
          { label: 'Pear', value: 'pear' },
          { label: 'Orange', value: 'orange' },
          { label: 'Grape', value: 'grape' },
          { label: 'Banana', value: 'banana' },
          { label: 'Fæðingarorlof', value: 'faedingarorlof' },
          { label: 'Atvinna', value: 'atvinna' },
          { label: 'Vottorð', value: 'vottord' },
        ])
      },
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={AsyncSelectFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<AsyncSelectFormField field={...} />` with the configuration object defined above.
