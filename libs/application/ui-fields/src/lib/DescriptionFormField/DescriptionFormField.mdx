# DescriptionFormField

### Usage in a template

You can create a DescriptionFormField using the following function `buildDescriptionField`.

<Source
  language="typescript"
  code={dedent(`
    buildDescriptionField({
      id: 'field.id',
      description: 'Field description',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={DescriptionFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<DescriptionFormField field={...} />` with the configuration object defined above.
