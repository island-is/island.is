# KeyValueFormField

### Usage in a template

You can create a KeyValueFormField using the following function `buildKeyValueField`.

<Source
  language="typescript"
  code={dedent(`
    buildKeyValueField({
      label: 'Label',
      value: 'Value',
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story name="Default">
    <KeyValueFormField
      application={createMockApplication()}
      field={{
        label: 'Label',
        value: 'Value',
      }}
    />
  </Story>
</Canvas>

You can also use this field into a custom component by using `<KeyValueFormField field={...} />` with the configuration object defined above.
