# ExpandableDescriptionFormField

### Usage in a template

You can create a ExpandableDescriptionFormField using the following function `ExpandableDescriptionFormField`.

<Source
  language="typescript"
  code={dedent(`
    ExpandableDescriptionFormField({
      id: 'field.id',
      title: 'Field title',
      introText: 'Field introText'
      description: '* bullet1\n * bullet2'
    })`)}
/>

The previous configuration object will result in the following component:

<Canvas>
  <Story of={ExpandableDescriptionFormFieldStories.Default} />
</Canvas>

You can also use this field into a custom component by using `<ExpandableDescriptionFormField field={...} />` with the configuration object defined above.
