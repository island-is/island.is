# CheckboxFormField

### Usage in a template

You can create a CheckboxFormField using the following function `buildCheckboxField`.

<Source
  language="typescript"
  code={dedent(`
    buildCheckboxField({
      id: 'field.id',
      title: 'Field title',
      width: 'half',
      options: [
        {
          value: 'phone',
          label: 'Phone',
          subLabel: 'Used to send notifications.',
        },
        {
          value: 'email',
          label: 'Email',
          subLabel: 'Used to send newsletters.',
        },
      ],
    })`)}
/>

The previous configuration object will result in the following component: This is the default UI recommended for the Checkbox field.

<Canvas>
  <Story of={CheckboxFormFieldStories.Default} />
</Canvas>

A more simple of choices should be displayed as follow:

<Canvas>
  <Story of={CheckboxFormFieldStories.ListOfCheckboxes} />
</Canvas>

You can also use this field into a custom component by using `<CheckboxFormField field={...} />` with the configuration object defined above.
