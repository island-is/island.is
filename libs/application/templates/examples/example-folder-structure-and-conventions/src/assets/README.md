# Assets

This folder is optional and can be used to store assets like images, icons, etc.

## Institution logos and Ísland.is graphics

Most assets stored in a shared folder for the application system to avoid duplication.

See the (/assets)[https://github.com/island-is/island.is/tree/main/libs/application/assets] folder

## What to store here

Assets that are unique to your application and will most likely not be used for other applications

## Naming and structure

assets should be named with camelCase, unless they are React components, then they should be named with PascalCase.

## Example

|-- assets/
|-- |-- myAsset.png
|-- |-- myAsset.svg
|-- |-- MyIcon.tsx
