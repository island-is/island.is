# car-rental-dayrate-returns

This template for an application was generated with the `nx generate-template car-rental-dayrate-returns` command.

This application is for car rental companies to post data on how many days a car that was registered to the day rate category was actually used and should be charged for.

To test this application, you an use the company 65°Artic through Gervimaður Færeyja in conjuction with the car-rental-fee-category.
Set some cars to use the day rate category in that application, then open this application and report the days for the same cars.

## Running unit tests

Run `nx test car-rental-dayrate-returns` to execute the unit tests via [Jest](https://jestjs.io).
