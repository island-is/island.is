# mile-car

This template for an application was generated with the `nx generate-template mile-car` command.

- What gervimenn can use this template
  Gervimaður Færeyjar has a mile car to test this application, once the car has been registered as a mile car, it can't be used again.

- What is this application for
  Registering cars that measure distance in miles instead of kilometers. This is done for a better user experience for those owners when registering the distance travelled on the car

- Which organization is this application for
  Data comes from SGS but the application is not for a specific organization

## Running unit tests

Run `nx test mile-car` to execute the unit tests via [Jest](https://jestjs.io).
