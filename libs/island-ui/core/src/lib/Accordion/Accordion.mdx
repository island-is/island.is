# Accordion

<DescriptionFigma name="Accordion" />

The **Accordion** component has three types of items:

- `<AccordionItem />` used inside an `<Accordion />` wrapper
- `<AccordionCard />` standalone card with a bordered container
- `<SidebarAccordion />` compact mini-style for sidebars

---

## Accordion + AccordionItem

Wrap `<AccordionItem />` elements inside `<Accordion />`. The `variant` prop on `<Accordion />` is inherited by all child items and controls the label size and icon size.

### Small (default)

`variant="small"` — h4 label, 40×40 icon.

### Single expand

With `singleExpand` (the default), only one item can be open at a time. Set `singleExpand={false}` to allow multiple items open simultaneously.

## AccordionCard

`<AccordionCard />` is a standalone bordered card. It does not require an `<Accordion />` wrapper. Use `variant` to choose between `"small"` (default) and `"large"`, and `colorVariant` for `"blue"` (default) or `"red"` theming.
