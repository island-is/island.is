# The Icelandic Recycling Fund Client (Úrvinnslusjóður)

This library was generated with [Nx](https://nx.dev).

## Running unit tests

Run `nx test clients-recycling-fund` to execute the unit tests via [Jest](https://jestjs.io).

## Running lint

Run `nx lint clients-recycling-fund` to execute the lint via [ESLint](https://eslint.org/).

## Usage

### Updating the open api definition (clientConfig.json)

```sh
yarn nx run clients-recycling-fund:update-openapi-document
```

### Regenerating the client:

```sh
yarn nx run clients-recycling-fund:codegen/backend-client
```
